# ClimateX 

ClimateX is a modern weather application that provides real-time weather updates, air quality information, and hourly forecasts for any location. With a clean and responsive interface, it delivers fast and accurate weather insights using API integration.

---

## 🚀 Features

- 🌍 Search weather by city name
- 📍 Get weather using current location
- 🌡️ Real-time temperature updates
- 💨 Air Quality Index (AQI) information
- ⛅ Hourly weather forecasts
- 📱 Responsive and user-friendly design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Weather API
